# Apple Global Product Sales Dashboard

[Live Demo](https://jeff-apple-project.streamlit.app)

Explore the world of Apple through its **global product sales**! This interactive Streamlit dashboard lets you dive into sales trends, compare product performance across regions, and uncover patterns in Apple’s market presence. Whether you're a data enthusiast, analyst, or tech fan, this project turns raw sales data into clear, engaging insights.

## Features

* **Interactive Visualizations:** Easily explore sales figures for Apple products like iPhone, iPad, Mac, and Apple Watch.
* **Regional Analysis:** Compare sales performance across countries and continents.
* **Time-Series Trends:** Track product sales trends over months and years.
* **Data Filtering:** Filter by product, region, or time period for customized insights.
* **User-Friendly Interface:** Built with Streamlit for a smooth, interactive experience.

## Technologies Used

* **Python** – Core programming language
* **Streamlit** – For building the interactive dashboard
* **Pandas & NumPy** – Data manipulation and analysis
* **Matplotlib / Seaborn / Plotly** – For visualizing trends and insights

## Purpose

The dashboard is designed to provide a clear overview of Apple’s global market presence and product sales dynamics. It’s perfect for anyone interested in **data analysis, business intelligence, or Apple product trends**.

## How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/xvm-swiss/ds_Session_12_Apple_Global_Product_Sales.git
   ```
2. Navigate to the project folder:

   ```bash
   cd /xvm-swiss/ds_Session_12_Apple_Global_Product_Sales
   ```
3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
4. Run the Streamlit app:

   ```bash
   streamlit run app.py
   ```
